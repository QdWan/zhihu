## 仿造实现知乎
## 技术栈
	前端: Less + Vue
	服务端: Express + Mongoose  
	数据库: MongoDB
## 功能
1. 提问,回答
2. 问题评论,回答评论,评论嵌套
3. 对问题,回答,评论进行赞踩 
4. 用户相互关注  
5. 用户操作时记录动态,并获取动态信息
